# Обход блокировок Discord и YouTube (portable)

Лёгкое портативное приложение для Windows, которое запускает обход DPI для Discord и YouTube и живёт в трее. Не требует установки и .NET — достаточно распаковать и запустить.

## Установка и запуск

1. Скачайте архив `obhod-portable.zip` из этого репозитория (или раздела Releases).
2. Распакуйте архив в любую папку (например, `C:\Obhod\`).
3. Запустите `TrayApp.exe`.
   - При первом запуске Windows спросит права администратора — согласитесь, иначе обход не заработает.

## Использование

- В трее появится иконка приложения (синяя рыбка).
- Откройте меню по правому клику:
  - Строка **`[ВКЛ.] / [ВЫКЛ.] Обход блокировок`** включает и выключает обход.
  - Пункт **«Запускать при старте Windows»** настраивает автозапуск через Планировщик задач.
  - Пункт **«Проверить обновления»** сверяет текущую версию с GitHub.
  - Пункт **«Выход»** полностью завершает приложение и отключает обход.

## Обновление

1. Закройте приложение через пункт **«Выход»** в трее.
2. Скачайте новую версию `obhod-portable.zip`.
3. Замените старые файлы распакованными файлами из нового архива.
4. Запустите `TrayApp.exe` снова.

> Поддерживаемые системы: Windows 10 / Windows 11 (x64)
